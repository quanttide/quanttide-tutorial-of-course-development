# 快速入门

我们主要是讲如何把实际生产中的真实情境变成一个可控的教学课堂 这节课我们以氛围编程为例。

我们要去了解编程小白面临的真实困难 比如现在工具的门槛其实已经越来越低了 大部分大模型网站都可以直接去生产；安装编程软件也有很多越来越简单的方法。作为编程小白 他们首先会面临搜索的困难 还会面临安装过程中各种繁琐步骤要怎么解决的问题 所以 我们帮助用户找到好方案 并帮助他们一步一步地去处理。

那么做课程的人会遇到什么困难呢 就是他们会面临那种比较常见的清识的体会 就是自己做着做明白 然后想也能想出来 但一到讲的时候 就感觉总感觉差了什么 有很多优秀的老师 在有具体的人反馈时 就可以去带入他的情景 而当做标准课时 在录制状态之下 就很难找到适合自己的方式 所以 我们需要从真实的情境之中去捕捉场景 然后围绕这些场景去描述。

是好的课程要放下那些隐的情 要能够把这些隐性的知识都给显性化。这需要一套系统的方法去感知 捕捉真实情景 把它描述成一个更合理的样子 我们在这种情境之下 一步步操作。
